How to Password Protect Your Files

Sending sensitive data like bank statements or contracts via email can be risky. If you send the file "raw," anyone who accesses the email can read it.

The solution is to lock the file with a password.

How encryption works
We wrap your file inside a "Secure ZIP." This acts like a digital safe. Without the correct password, the content inside is scrambled and unreadable.

Why use ZIP encryption?

  • Universal Access: Windows and Mac computers can open secure ZIP files without installing new software.
  • Safe Sharing: You can email the file safely. Just make sure to send the password through a different channel (like a text message).
  • Any File Type: You can lock PDFs, images, Word docs, or Excel sheets.

How to Lock a File

  1. Open the Locker: Go to our Secure File Locker tool.
  2. Upload: Select the file you want to protect.
  3. Set Password: Type a strong password. Make sure you remember it!
  4. Download: Click "Create Secure ZIP" to get your locked file.

Extract Pages from PDF Online

Have a large PDF document but only need a few specific pages? The Image Toolkit's PDF Splitter allows you to visually select pages and extract them into a new file instantly.

Split Modes Explained

  1. Extract Mode: Use this when you want to save specific pages into a single new document. For example, extracting just the Invoice page from a 10-page contract.
  2. Separate Mode: This creates a separate PDF file for each page you select. If you select 5 pages, you will download a ZIP file containing 5 individual PDFs.

Features

  • Visual Selection: No need to remember page numbers. See thumbnails of every page and click to select.
  • Secure: We use client-side technology (PDF-Lib). Your document is processed in your browser and never uploaded to a server.
  • Fast: Since there are no uploads, even large 100MB files open instantly.
